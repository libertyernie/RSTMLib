RSTMLib is a subset of BrawlLib that contains only the RSTM (.brstm) encoder and its dependencies.

The Encode method in RSTMEncoder has been changed to EncodeToByteArray, and the output is of type byte[] instead of FileMap (the FileMap type is not included.)
